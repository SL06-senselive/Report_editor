# **App Name**: Energy Audit Pro

## Core Features:

- Text Field Editing: Enable inline editing of all text fields within the report, such as titles, metadata, KPIs, and remarks.
- Image Upload and Drag-and-Drop: Implement drag-and-drop and upload functionality for all chart slots in the report.
- Image and Text Composer: Integrate a React-based image composer, where users can upload an image and add customizable text overlays. This feature replicates the existing functionality of the composer. The generated composer image can be saved on the report
- PDF Report Generation: Generate a multi-page PDF report capturing the entire content of the report, including all edited text and uploaded images.
- Report Reset: Implement a 'Reset All' button that reverts all text fields and images to their default state.
- AI-Powered Insight Tool: Integrate an AI tool that analyzes the uploaded charts and entered data to suggest relevant insights for the report's conclusion section, identifying potential issues or areas of improvement.

## Style Guidelines:

- Primary color: Deep blue (#1E4D8C), mirroring the original design's primary color to maintain brand recognition and trust.
- Background color: Light gray (#F6F7FB), providing a clean, professional backdrop that ensures readability and visual appeal.
- Accent color: Red-orange (#E74C3C), used for error messages and destructive actions like clearing an image slot.
- Body and headline font: 'Inter' (sans-serif) for clear, readable text throughout the application.
- Utilize simple, consistent icons to represent actions and sections within the report, enhancing usability and visual coherence.
- Maintain the original layout's structure with clear sections and card-based organization to ensure familiarity and ease of navigation for existing users.
- Incorporate subtle animations and transitions for user interactions, such as image uploads or text edits, to provide a smooth and engaging user experience.
